# Math Recall

A React Native mobile application for practicing basic math operations. Built with Expo for iOS, Android, and web platforms.

## Features

- **Addition Practice** - Improve your addition skills
- **Subtraction Practice** - Master subtraction problems
- **Multiplication Practice** - Build multiplication fluency

Customize your practice sessions by:
- Selecting a specific number to focus on (1-9)
- Choosing the number of questions (10, 20, or 30)

## Screenshots

The app features a clean, modern interface with:
- Smooth animated transitions between screens
- Responsive layout that adapts to different screen sizes
- Color-coded practice types for easy identification

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn
- Expo CLI (optional, but recommended)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd math-recall
```

2. Install dependencies:
```bash
npm install
```

### Running the App

Start the development server:
```bash
npm start
```

Then choose your platform:

**iOS Simulator** (macOS only):
```bash
npm run ios
```

**Android Emulator**:
```bash
npm run android
```

**Web Browser**:
```bash
npm run web
```

You can also scan the QR code with the Expo Go app on your mobile device.
